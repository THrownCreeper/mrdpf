Command Line Interface¶

usage: mrdpf_cli [-h] (-d DIR | -f FILE) -o OUT [--no-recurse] [--clear]

Named Arguments¶

-d, --dir

Directory to search for compatible files

-f, --file

File to parse

-o, --out

Folder to write results to

--no-recurse

Do not recurse. If input option is not –dir this option is ignored.

Default: False

--clear

Clear output directory before writing results (will delete all files below out directory)

Default: False
